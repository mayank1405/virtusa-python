WEATHER DATA ANALYZER AND FORECAST DASHBOARD

Greetings !!

This project aims to help users get meaningful weather related data at the click of a button.stead of searching across multiple websites, 
users can view weather data, analyze trends, and generate basic forecasts from a single dashboard.

Features:

* We fetch weather data using APIs to provide accurate weather data.
* Store weather data locally.
* Tools like matplotlib to help visualize trends like temperature
* We also provide simple prediction based on past weather data using simple regression
* All this made accessible using a clean and simple CLI dashboard

Technologies used:

* Python
* Requests
* Matplotlib
* Numpy
* Scikit Learn
* Weather API
